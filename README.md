# 涉外法治大模型智能服务平台 Demo

基于 Vue3 开发的涉外法治大模型智能服务平台前端展示 Demo。

## 项目特性

- ✅ Vue3 + Vite 现代化前端框架
- ✅ 响应式设计，支持多端适配
- ✅ 五大核心功能模块
- ✅ 美观的渐变色 UI 设计
- ✅ 静态数据模拟展示

## 功能模块

### 1. 多语种涉外法律咨询服务
- 支持中、蒙、俄、英四大语种
- 智能对话交互
- 常见问题快速咨询
- 参考依据展示

### 2. 跨法域涉外法律智能检索服务
- 基础精准检索
- 高阶要件检索
- 多维度筛选
- 检索结果详情展示

### 3. 涉外法律动态监测、风险预警与决策支撑
- 常态化动态监测
- 趋势预警与风险评估
- 专项分析与决策建议
- 风险趋势图表

### 4. 涉外法律文书智能处理与生成
- 多种文书类型支持
- 智能解析与信息提取
- 文书模板库
- 历史记录管理

### 5. 涉外合同智能审查与跨境合规风险防控
- 多种合同类型
- 多维度审查
- 风险分级识别
- 修订建议生成

## 安装与运行

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 技术栈

- Vue 3.4.0
- Vue Router 4.2.5
- Pinia 2.1.7
- Vite 5.0.0

## 项目结构

```
overseasLLM2/
├── src/
│   ├── components/       # 组件
│   │   └── Header.vue   # 头部导航组件
│   ├── views/           # 页面
│   │   ├── Consultation.vue    # 智能咨询
│   │   ├── Search.vue          # 法律检索
│   │   ├── Monitor.vue         # 动态监测
│   │   ├── Document.vue        # 文书处理
│   │   └── Contract.vue        # 合同审查
│   ├── router/           # 路由
│   │   └── index.js
│   ├── App.vue          # 根组件
│   ├── main.js          # 入口文件
│   └── style.css        # 全局样式
├── index.html           # HTML 模板
├── package.json         # 项目配置
└── vite.config.js      # Vite 配置
```

## 注意事项

- 本项目为前端展示 Demo，所有数据均为静态模拟数据
- 实际生产环境需要对接后端 API
- 部分功能（如文件上传、报告导出等）为模拟实现
